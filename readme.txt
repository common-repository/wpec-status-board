Integrate the WP e-Commerce sales logs with the Status Board iPad app from Panic.

== Description ==

WP e-Commerce - Status Board Integrates the WP e-Commerce Sales Logs with the Status Board iPad app.

You can add the graphs to Status Board via the Button or manually with information in the contextual help on the default WP e-Commerce Sales Logs page in wp-admin.

= The following filters exist =
* wpecsb_output - Alters the output after it's been formatted
* wpec_statusboard_date_format - Alter the date format of the X-Axis. (Default: n/j or, month/day, uses PHP date formatting)

== Installation ==

1. Install the WP e-Commerce - Status Board plugin
2. Activate the plugin
3. Add the graph via the Sales Logs page
4. Enjoy Status Board Updates for your WP e-Commerce Site
